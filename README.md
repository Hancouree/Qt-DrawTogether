# DrawTogether

## Description

DrawTogether is a desktop application that allows multiple users to draw together in real time.
The main goal of the project is to create a shared drawing space where you can draw simultaneously
with your girlfriend, friends, or other users, see each other's strokes instantly, and save the
created drawings.

The application focuses on real-time collaboration, simplicity, and an intuitive user interface.

---

## Features

- Real-time collaborative drawing
- Multiple users connected at the same time
- Live synchronization of drawing strokes
- Saving drawings locally
- Desktop application

---

## Technologies Used

- **C++**
- **QML (Qt Quick)** for the user interface
- **Qt Framework**
- **QWebSocket** for real-time communication between clients

---

## Project Goal

The goal of this project is to explore real-time networking and UI development using Qt,
as well as to build a practical and fun application for collaborative creativity.
